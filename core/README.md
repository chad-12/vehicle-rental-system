# Vehicle Rental System (Vibe Coding Project)

## Project Description
This is a simple Vehicle Rental System built using Django. It allows users to view available vehicles and manage them through admin panel.

## Features
- Add vehicles via admin panel
- View available vehicles on homepage
- Basic rental system structure
- Django admin integration

## Tech Stack
- Django
- Python
- SQLite
- HTML

## How to Run Project

```bash
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
