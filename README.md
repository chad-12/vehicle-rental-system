\# Vehicle Rental System (Vibe Coding Project)



A simple Django-based vehicle rental system.



\## Features

\- View vehicles

\- Add vehicles (admin)

\- Simple rental structure



\## Tech Stack

\- Django

\- Python

\- SQLite

\- HTML



\## Run Project



pip install django

python manage.py runserver



Open:

http://127.0.0.1:8000/



Admin:

http://127.0.0.1:8000/admin
